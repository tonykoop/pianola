# Assembly Sequence

Status: planning sequence only. No jig dimensions or fabrication coordinates
are released.

## Phase A: Reference And Test Media

- Select or create non-authority roll test strips.
- Assign IDs to roll media, tracker-bar mockups, pouches, valves, and action
  rigs.
- Prepare a measurement log before any parts are declared successful.

Hold point: do not fabricate a full tracker bar until the one-hole rig has
validated paper tracking and sensing.

## Phase B: Tracker-Bar Coupon

- Build or mock up a single-port tracker-bar section.
- Provide transparent or removable inspection access.
- Add leak-isolation connection points.
- Test paper motion, port exposure, sealing, and cleaning.

Suggested jigs:

- Paper alignment jig with adjustable guides.
- Tracker-bar clamping jig for repeatable paper contact.
- Leak-test adapter for closed and open states.

## Phase C: Pouch And Valve Bench Rig

- Assemble one pouch and one valve as a serviceable module.
- Add visible vent and leak-isolation paths.
- Test response, sticking, reset, and replacement.

Suggested jigs:

- Pouch clamping fixture.
- Valve-seat inspection fixture.
- Leak isolation manifold.

## Phase D: Single-Note Action Rig

- Couple the tracker coupon to the pouch/valve module.
- Couple the pneumatic output to a representative action interface.
- Add a safe decoupler before using any real piano action.
- Record missed triggers, slow resets, and stuck-open behavior.

Suggested jigs:

- Action-interface mock rail.
- Adjustable actuator mount.
- Non-destructive action stop.

## Phase E: Scale-Up Readiness Review

- Review whether failure modes are isolated by subsystem.
- Decide whether tracker, pneumatic, and action interfaces have enough evidence
  for a larger note group.
- Update the visual-output register only if measured evidence exists.

Hold point: do not add expression or sustain mechanisms until the basic
single-note chain is repeatable.
